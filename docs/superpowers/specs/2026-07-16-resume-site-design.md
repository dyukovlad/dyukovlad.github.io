## Техника

- Файлы в корне репозитория: `index.html`, `styles.css`, `script.js`, `resume.pdf`
  (копия PDF пользователя). Папка `budgety/` остаётся нетронутой.
- Палитра: фон `#0b0f14`, светлый текст, акцент — неоново-бирюзовый (~`#39d1b4`),
  приглушённый серо-голубой для второстепенного текста.
- Шрифты: системный стек + моноширинный стек для акцентов. Без внешних загрузок.
- Эффекты (прогрессивное улучшение, всё работает и без JS):
  - canvas-частицы в hero (пауза при `prefers-reduced-motion`, невидима для скринридеров);
  - typing-эффект строки ролей (без JS отображается статичная первая роль);
  - появление секций при скролле через IntersectionObserver;
  - счётчики метрик при попадании в viewport.
- vCard генерируется на клиенте (как в текущей версии), телефон копируется по клику.
- `@media print`: светлая тема, скрыты canvas и кнопки — страница печатается как резюме.
- SEO: `meta description`, Open Graph (title/description/url), favicon — inline SVG,
  семантические теги (`header/main/section/footer`), `lang="ru"`.
- Адаптивность: одна колонка на мобильных, сетка на десктопе; горизонтального
  скролла нет.

## Ошибки и крайние случаи

- GitHub API недоступен/лимит — карточки проектов показываются без звёзд.
- JS отключён — весь контент доступен, эффекты не запускаются.
- `prefers-reduced-motion` — анимации отключены.

## Проверка

- Локальный запуск: `python3 -m http.server` из корня репозитория.
- Проверить: десктоп и мобильная ширина, печать (Cmd+P), отсутствие
  горизонтального скролла, работу кнопок PDF/vCard/копирования телефона.

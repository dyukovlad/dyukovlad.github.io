Подсказки
  • git init - создает репозиторий на основе папки.
  • git status - проверяет наличие изменений после последнего комита.
  • git commit -m "//Комментарий" - сохраняет изменения в текущей ветке.
  • git push - сохраняет изменения на удаленный репозиторий.
  • git pull - получить изменения с удаленного репозитория.
  • git log - история всех commit.
  • git branch -a - список всех веток проекта.
  • git clone - получить новый проект с удаленного репозитория.
  • git checkout -f - откатиться к флагу.

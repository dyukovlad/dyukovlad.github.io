Подсказки
9 самых важных команд для GIT:

1. git init - создает репозиторий на основе папки.
2. git status - проверяет наличие изменений после последнего комита.
3. git commit -m "//Комментарий" - сохраняет изменения в текущей ветке.
4. git push - сохраняет изменения на удаленный репозиторий.
5. git pull - получить изменения с удаленного репозитория.
6. git log - история всех commit.
7. git branch -a - список всех веток проекта.
8. git clone - получить новый проект с удаленного репозитория.
9. git checkout -f - откатиться к флагу.
